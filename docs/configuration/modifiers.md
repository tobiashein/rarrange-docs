# Modifiers

A sorter can have one or multiple assigned additional _modifiers_ to further define the sort order of elements.

| Name       |
|------------|
| `abstract` |
| `async`    |
| `const`    |
| `event`    |
| `extern`   |
| `new`      |
| `override` |
| `partial`  |
| `readonly` |
| `sealed`   |
| `static`   |
| `unsafe`   |
| `virtual`  |
| `volatile` |
